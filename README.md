## 如何运行：

1. 直接运行Library底部的main方法，按照程序提示输入你的操作指令。
2. 特别需要注意的是所有的符号如“,”和“:”都是英文符号
3. 为了测试方便，我为你初始化了三个Student，方便你执行“2.生成成绩单”，它们的学号分别是:1,2,3。
